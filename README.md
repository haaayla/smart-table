# sp6-2_smart-table_starter

# Smart Table 1.0

Модульная реализация "умной таблицы" на чистом JavaScript.  
Проект демонстрирует архитектурный подход к обработке данных через композицию независимых модулей.

## 🔗 Репозиторий

https://github.com/haaayla/smart-table

---

## 🚀 Возможности

- Поиск по нескольким полям (customer, seller, date, id)
- Фильтрация по выбранным значениям
- Сортировка по колонкам
- Пагинация с динамическим пересчётом страниц
- Сброс фильтров стандартным reset формы
- Единый pipeline обработки данных

---

## 🧠 Архитектура

Обработка данных построена по принципу последовательной трансформации:

data
→ searching
→ filtering
→ sorting
→ pagination
→ render


Каждый шаг реализован как независимый модуль:

- `initSearching`
- `initFiltering`
- `initSorting`
- `initPagination`

Модули принимают данные и состояние формы, возвращают преобразованный массив.  
События формы обрабатываются централизованно через `render(action)`.

---

## 🛠 Используемые технологии

- JavaScript (ES Modules)
- Vite
- HTML5 / CSS3
- DOM API
- FormData
- Модульная архитектура без фреймворков
